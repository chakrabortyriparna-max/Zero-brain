# Dark Factory Mission

## Purpose
The Dark Factory is an autonomous AI code-factory that maintains and evolves this repository without human intervention.

## Goals
1. Detect issues, bugs, and maintenance tasks via the orchestrator
2. Triage incoming work against defined rules
3. Implement accepted issues autonomously
4. Validate all changes with a holdout review before merging

## Scope
- Automated issue triage and classification
- Autonomous PR creation and review
- Weekly comprehensive regression testing
- Self-healing when tests fail

## Non-Goals
- Feature requests requiring product decisions
- Breaking architectural changes without human approval
- Direct deployment to production without review
